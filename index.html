What is React?

React is a JavaScript library created by Facebook. React is a tool for building reusable UI components.

How does React Work?

React creates a VIRTUAL DOM in memory.

Instead of manipulating the browser's DOM directly, React creates a virtual DOM in memory, where it does all the necessary manipulating, before making the changes in the browser DOM.

React creates a VIRTUAL DOM in memory.

This process improves the performance exponentially and in React terminology this entire process is known as Reconciliatory Algorithm.

The Virtual DOM works in three simple steps.

Whenever any underlying data changes, the entire UI is re-rendered in Virtual DOM representation.

Compare the differences

Then the difference between the previous DOM representation and the new one is calculated.

Update the Real DOM

Once the calculations are done, the real DOM will be updated with only the things that have actually changed.
